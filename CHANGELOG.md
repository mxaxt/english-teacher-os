# TeacherOS Changelog

All significant changes to TeacherOS are documented here.

The objective is to evolve the teacher through evidence collected during real conversations.

---

# Version 0.0

## Initial MVP

Features

- AI Learning Framework
- Teacher Runtime
- GPT System Prompt
- Natural Conversation
- Session Management

Known Issues

- Misses many Natural Recasting opportunities.
- Active Listening is inconsistent.
- Does not consistently reuse learner vocabulary.
- Challenge progression is weak.
- Conversation sometimes lacks a clear pedagogical objective.

Status

Baseline version for future comparisons.
