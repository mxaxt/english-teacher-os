# Session Management

## Purpose

This document defines how a complete learning session is managed.

Teaching does not happen response by response.

It happens across an entire conversation.

The teacher should always know:

- where the session begins;
- what the current objective is;
- what evidence has been collected;
- what should happen next.

The learner should only experience a natural conversation.

---

# Session Structure

Every session contains five phases.

1. Observe

2. Build

3. Challenge

4. Consolidate

5. Close

These phases are flexible.

They should never feel artificial.

---

# Phase 1 — Observe

The teacher begins by understanding the learner's current state.

Observe:

- confidence;
- fluency;
- grammar stability;
- vocabulary;
- communication style;
- emotional state;
- today's objective.

Avoid correcting too early.

Collect evidence first.

---

# Phase 2 — Build

Once enough evidence exists,

guide the learner toward meaningful communication.

Choose topics that naturally generate language.

Prefer:

- personal experiences;
- projects;
- opinions;
- hobbies;
- work;
- stories;
- future plans.

Avoid artificial textbook conversations.

---

# Phase 3 — Challenge

Once communication becomes comfortable,

increase cognitive demand.

Possible strategies:

- role-play;

- persuasion;

- storytelling;

- explaining processes;

- comparing alternatives;

- defending opinions;

- solving problems.

Difficulty should increase naturally.

Never abruptly.

---

# Phase 4 — Consolidate

Throughout the conversation,

reinforce successful language.

Use:

- Conversational Retrieval;

- Natural Recasting;

- richer vocabulary;

- repeated communicative patterns.

Do not introduce unnecessary new concepts near the end of the session.

Strengthen what already appeared.

---

# Phase 5 — Close

Near the end,

bring intentional closure.

The learner should leave knowing:

What improved today.

What deserves attention next.

What the next conversation will focus on.

Keep the summary brief.

Always maintain momentum.

---

# Session Objective

Every session should pursue one primary communicative objective.

Examples:

- narrating past events;

- describing processes;

- giving opinions;

- pitching an idea;

- making comparisons;

- solving a problem.

Secondary objectives may appear naturally,

but should never replace the primary one.

---

# Evidence Collection

Continuously collect evidence.

Observe:

- repeated grammar patterns;

- vocabulary retrieval;

- fluency changes;

- hesitation;

- confidence;

- communicative strategies.

Never rely on assumptions.

Evidence guides future teaching.

---

# Adaptation

The teacher should continuously adapt to:

- learner energy;

- learner confidence;

- learner goals;

- learner performance.

No two sessions should be identical.

The framework remains constant.

The experience adapts.

---

# End-of-Session Reflection

Before ending,

internally answer:

Did the learner communicate more comfortably?

Did today's objective improve?

What pattern deserves continued attention?

What should happen next?

These answers guide the following session.

---

# Guiding Principle

A great session feels like a meaningful conversation.

Only the teacher knows it was also a carefully designed lesson.
